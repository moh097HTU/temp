Flight Controller

Connected via UART

Video Channel (UDP Unicast)
Jetson → QGC Port 5600
RTP Stream (H.264)
  • Source: OAK-D Camera
  • Resolution: 1280x720 @ 30fps (Configurable)
  • Protocol: GStreamer/UDP
MAVLink Telemetry (UDP)
Jetson → QGC Port 14550
NAMED_VALUE_INT / FLOAT
  • BAT1/2_ACTIVE: Battery GPIO Status
  • TRK_COUNT: Number of tracked objects
  • TRK_LOCKED: 1 if target locked, else 0
  • TRK_LOCK_ID: ID of locked target
  • TRK_YAW/PIT_ERR: Tracking error angles
MAVLink Commands (UDP)
QGC → Jetson Port 14551
COMMAND_LONG (User Range)
  • 31100: START_TRACKING
  • 31101: STOP_TRACKING
  • 31102: SELECT_TARGET_ID (param1=ID)
  • 31103: SELECT_PIXEL (param1=X, param2=Y)
Standard Telemetry (Pass-through)
Flight Controller → Jetson → QGC
Standard MAVLink
  • HEARTBEAT: System status & Mode
  • SYS_STATUS: Main battery voltage/current
  • GPS_RAW_INT: Position data
  • ATTITUDE: Roll/Pitch/Yaw
